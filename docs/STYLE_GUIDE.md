## 🔤 2. Tipografía Recomendada

Se han seleccionado fuentes de **Google Fonts** para facilitar la implementación técnica y asegurar la compatibilidad.

### **Primaria: Quicksand** (Para Títulos y Botones)

- **Estilo:** Rounded Sans Serif  
- **Por qué:** Sus bordes redondeados transmiten amabilidad, empatía y suavidad, ideal para el mundo de las mascotas.  
- **Pesos sugeridos:** Bold (700) para títulos; Medium (500) para botones  

### **Secundaria: Montserrat** (Para Cuerpo de Texto)

- **Estilo:** Geometric Sans Serif  
- **Por qué:** Alta legibilidad en pantallas pequeñas y un aspecto muy profesional/tecnológico  
- **Pesos sugeridos:** Regular (400) para descripciones; Light (300) para textos secundarios  

### **Jerarquía Visual (Escala)**

1. **H1 - Títulos de Sección:** Quicksand Bold | 24px  
2. **H2 - Nombres de Mascotas:** Quicksand Medium | 18px  
3. **Cuerpo de texto:** Montserrat Regular | 14px - 16px  
4. **Captions / Leyendas:** Montserrat Light | 12px  

---

## 🛠️ 3. Regla de Implementación 60-30-10

Para mantener el equilibrio visual en la app:

- **60% (Base):** `Blanco Off-white` (Fondos y espacios limpios)  
- **30% (Identidad):** `Turquesa Digital` y `Verde Menta` (Botones, iconos, marca)  
- **10% (Acento):** `Azul Petróleo` (Lectura) y `Coral/Amarillo` (Alertas y llamadas de atención)
